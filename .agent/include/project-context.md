# Project Context

## What This Repository Is

- Repository: `agam-lang/agam-lang.github.io`
- Purpose: Provide the official documentation website for the Agam language including language guide, API reference, tutorials, and blog.
- Technology: Static site (mdBook or Docusaurus)

## Organization Context

- This is part of the `agam-lang` ecosystem.
- The core Agam compiler lives at `agam-lang/agam`.
- The standard library lives at `agam-lang/std`.
- The scientific platform lives at `agam-lang/agamlab`.

## Key Areas

- Language Guide
- API Reference
- Tutorials
- Blog
- Getting Started
